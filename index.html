WordPress

O que é?

O WordPress é um sistema de gerenciamento de conteúdo utilizado para auxiliar na criação de sites e práticas. Em casos específicos, seu objetivo é facilitar a criação de conteúdo sem a necessidade de utilizar linguagem de programação. Ele visa possibilitar que até mesmo criadores sem conhecimento em códigos possam gerar uma loja ou blog de forma simples e fácil. Além disso, o WordPress oferece a opção de personalizar layouts e outras funcionalidades. A proposta do WordPress é democratizar o desenvolvimento de blogs, profissionais e grandes sites.

O surgimento do WordPress

O surgimento começou em 2002 de uma forma bem inusitada. Matt Mullenweg, um estudante de ciências políticas dos EUA, começou a utilizar um sistema de blogging criado por Mike Little. Esse projeto feito pela GNU era conhecido como B2 ou cafelog. Matt se tornou um usuário assíduo do sistema, o que lhe causou frustração quando descobriu que o projeto não teria mais continuação por decisão do criador. Ele postou em um blog uma mensagem lamentando a falta de atualização do B2 e ressaltando que, como o sistema tinha o código aberto, era possível criar uma variação do projeto. É aí que tudo acontece: sem nem ter contato com Matt Mullenweg, Mike respondeu ao post se colocando à disposição para ajudar no projeto, caso fosse algo sério. A mensagem original foi deixada por Mike em 25/01/2003.

Conclusão

Conclusão: Sobre esse assunto, o WordPress começou como uma plataforma de criação e gerenciamento de blogs, mas com o passar do tempo, ele se tornou um sistema de desenvolvimento para todos os tipos de site, desde os projetos simples até os mais complicados. O WordPress se destacou por vários fatores, entre eles seu módulo intuitivo para as pessoas que não entendem de código. As chances de você gerenciar o conteúdo de um site baseado em WordPress são altas.
